# Task Manager

Fast Window v5 任务记录与管理应用。

## 功能根系

- 主页用方形卡片表示任务分组。
- 点击卡片进入任务分组页面，一行一个任务条目。
- 顶部栏加号创建任务分组。
- 任务分组页面提供新建任务入口。
- 创建弹窗支持按钮保存和 Ctrl+S 保存。
- 任务分组页面支持直接 Ctrl+V 粘贴文本新建任务条目。
- 任务条目以卡片展示，点击卡片打开任务详情。
- 任务详情默认只读查看，点击编辑后可以修改并保存。

## 技术边界

- 任务数据由本地应用数据目录承载，前端只负责展示、创建和编辑交互。
- 窗口、托盘、单实例、后台启动与数据目录检查由桌面壳负责。
- 本地后台负责任务分组和任务条目的读写，并通过本地连接向界面提供能力。
