### 界面重写

| 项目 | 说明 |
| --- | --- |
| 页面入口 | 顶部栏提供收藏夹、全部笔记入口 |
| 维护入口 | 归档和回收站移动到设置页面内部 |
| 全部笔记 | 全部笔记只展示正常笔记，不混入归档和回收站 |
| 收藏夹范围 | 收藏夹按收藏内容展示，不沿用全部笔记的正常笔记范围 |
| 页面瘦身 | 删除页面内部大型页头，连接状态和刷新入口收敛到顶部栏 |
| 卡片展示 | 全部笔记、归档、回收站和收藏夹内部笔记统一使用卡片展示 |
| 笔记弹窗 | 点击笔记卡片后使用模态窗展示完整笔记内容 |
| 详情区域 | 移除页面内嵌笔记详情区，避免占用页面内容空间 |
| 收藏夹 | 收藏夹以卡片展示，进入父收藏夹后可继续进入子收藏夹 |
| 收藏夹路径条 | 收藏夹页面顶部显示当前位置，并支持点击路径跳到任意上级 |
| 页面位置 | 每个页面和收藏夹内部位置独立记忆，切换后保留当前位置 |
| 视觉风格 | 移除渐变色风格，改为纯色、边框和卡片布局 |
